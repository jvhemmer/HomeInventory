# Home Inventory Mod for Project Zomboid

## Overview
The Home Inventory Mod enhances the gameplay experience in Project Zomboid by adding a user-friendly interface that allows players to view all items stored in a designated area of their base. This mod introduces a button next to the crafting, character, and building icons, which opens a table displaying the inventory when clicked.

## Features
- **Inventory Button**: A new button is added to the UI for easy access to the inventory.
- **Item Display**: When the button is clicked, a table appears showing all items in the specified area of the player's base.
- **User Interface**: The mod includes a clean and organized layout for displaying items, making it easy for players to manage their inventory.

## Installation
1. Download the Home Inventory Mod files.
2. Place the `HomeInventoryMod` folder into the `mods` directory of your Project Zomboid installation.
3. Launch Project Zomboid and enable the Home Inventory Mod from the mods menu.

## Usage
- After installation, start a game and look for the new inventory button next to the crafting, character, and building icons.
- Click the button to open the inventory table and view all items in the designated area of your base.

## Compatibility
This mod is designed for Project Zomboid version B42. Ensure that your game is updated to this version for optimal performance.
